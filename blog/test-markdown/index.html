git工作笔记

增加index1页面，强制设置首页提醒

配置git在命令行中着色

git config --global color.ui true

获取reflog

git reflog

使用ksdiff按git版本提取参照

git difftool 71e722a 333867c

比对差异

ksdiff index.html index1.html

添加所有文件至git跟踪

git add .

快捷提交

git commit -a -m "版本说明"

导出纯净版本

git archive master | tar -x -C ../output
